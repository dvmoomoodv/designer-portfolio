# Rohigraphy Archive Portfolio

English documentation: [README.md](./README.md)

라이브 사이트: [http://rohigraphy.co.kr:63478/](http://rohigraphy.co.kr:63478/)

비주얼 크리에이터, 사진가, 연구자, 콘셉트 아티스트를 위한 가벼운 PHP 기반 포트폴리오 CMS입니다.

Rohigraphy Archive Portfolio는 사진, 연구, 콘셉트 아트, 스토리보드, 미디어문화연구를 함께 다루는 PHP 기반 비주얼 포트폴리오이자 아카이브 웹사이트입니다. 이 프로젝트는 범용 템플릿이 아니라, 스냅사진 작가이자 여러 시각 작업을 함께 다루는 한 개인과 하나의 개인 브랜드를 위해 제작되었습니다.

이 프로젝트는 단순한 정적 포트폴리오 페이지가 아닙니다. 가벼운 어드민 대시보드를 포함하고 있으며, JSON 데이터 구조를 통해 콘텐츠 수정, 이미지 슬롯 관리, 프로젝트 업데이트, 다국어 포트폴리오 구성, 페이지 디자인 설정, SEO 설정, 아카이브 관리를 할 수 있습니다.

Rohigraphy Archive Portfolio는 사진, 다큐멘터리 작업, 콘셉트 아트, 연구 노트, 데이터 플루언시, 미디어문화연구를 하나의 편집 가능한 아카이브로 묶는 가벼운 포트폴리오 CMS입니다.

현재 버전은 배포와 운영을 쉽게 하기 위해 PHP 템플릿과 JSON 콘텐츠 파일을 사용합니다. 추후에는 검색, 필터링, 미디어 관리, 장기적인 콘텐츠 운영을 위해 웹 데이터베이스 기반 구조로 전환할 예정입니다.

## 개요

Rohigraphy Archive Portfolio는 한 명의 다분야 창작자를 위한 개인 비주얼 아카이브입니다. 스냅사진, 자연사진, 다큐멘터리 이미지, 콘셉트 아트, 연구 노트, 문화연구, 데이터 플루언시, 질적 연구, 드로잉, 애니메이션 관련 시각 개발 작업을 하나의 포트폴리오 플랫폼으로 묶습니다.

목표는 무거운 CMS에 의존하지 않으면서도 깨끗하고 유지보수 가능하며 실제 운영 가능한 포트폴리오 시스템을 만드는 것입니다.

## 이 프로젝트의 의미

많은 포트폴리오 웹사이트는 정적으로 만들어져서 런칭 이후 유지보수가 어렵습니다. 이 프로젝트는 시간이 지나도 실제로 운영할 수 있는 포트폴리오를 목표로 합니다.

콘텐츠와 화면 구조를 JSON 데이터로 분리하고, 비개발자도 수정 가능한 어드민 인터페이스를 제공합니다. 따라서 디자이너 포트폴리오, 사진가 포트폴리오, 스냅사진 아카이브, 비주얼 리서치 아카이브, 콘셉트 아트 포트폴리오, 개인 브랜드 웹사이트, 소규모 크리에이티브 스튜디오 웹사이트에 적합합니다.

## 주요 기능

- 반응형 포트폴리오 웹사이트
- PHP 기반 페이지 렌더링
- 가벼운 어드민 대시보드
- JSON 기반 콘텐츠 관리
- 이미지 업로드 및 교체 지원
- 폰트 업로드 지원
- 다국어 콘텐츠 구조
- 프로젝트 카드와 프로젝트 상세 페이지
- 페이지별 / 전체 공통 디자인 설정
- 데이모드, 다크모드, 헤더 색상 설정
- 홈 화면 레이아웃 설정
- 아카이브 페이지별 페이징 설정
- SEO 메타데이터 설정
- LLM이 읽기 쉬운 `llm.txt` 제공
- 재사용 가능한 include와 공통 레이아웃 컴포넌트
- PHP 지원 서버에서 간단히 배포 가능

## 사이트 섹션

- `Home` — 메인 소개, 히어로 섹션, selected works, 카테고리, notes
- `About Me` — 창작자의 스토리, 교육, CV, 배경 정보
- `Concept Art` — 콘셉트 디자인, 스토리보드, 영화/애니메이션 중심 시각 작업
- `Research` — 문화연구, 미디어연구, 데이터 사이언스, 데이터 플루언시, 질적 연구 아카이브
- `Photography` — 다큐멘터리, 자연사진, 스냅사진, 프로필, 커플, 가족, 시각 레퍼런스 이미지
- `Doodle & Scribble` — 러프 스케치, 시각 노트, 비공식 드로잉, 초기 아이디어
- `Projects` — 포트폴리오 프로젝트 카드와 프로젝트 상세 페이지
- `Resume` — 경력, 역량, 프로필, 연락 정보
- `Contact` — 문의 정보, 소셜 링크, 브랜드 정보, 협업 레퍼런스

## 어드민 대시보드

이 프로젝트는 주요 웹사이트 콘텐츠를 관리하기 위한 어드민 대시보드를 포함합니다.

수정 가능한 영역은 다음과 같습니다.

- 사이트 공통 설정
- 네비게이션 메뉴와 드롭다운
- 푸터
- 메타데이터와 SEO 키워드
- Home, About, Work, Research, Photography, Doodle, Resume, Contact 콘텐츠
- 프로젝트 목록과 상세 데이터
- 이미지와 visual item 슬롯
- 폰트
- 데이모드와 다크모드 색상
- 헤더 색상
- 홈 레이아웃 설정
- 아카이브 페이징 기준

이미지는 업로드 후 콘텐츠 슬롯에 바로 연결할 수 있습니다. 폰트도 업로드하여 디자인 설정에서 적용할 수 있습니다.

## 프로젝트 구조

```txt
designer-portfolio/
├── admin/          # 어드민 대시보드와 콘텐츠 편집기
├── assets/         # CSS, JavaScript, 이미지, 업로드 자산
├── data/           # JSON 기반 콘텐츠 데이터
├── html/           # 기존 정적 HTML 아카이브 파일
├── includes/       # 공통 PHP 컴포넌트와 레이아웃 파일
├── index.php       # 메인 홈 페이지
├── about.php       # About 페이지
├── work.php        # Concept Art 페이지
├── research.php    # Research 아카이브 페이지
├── photograph.php  # Photography 페이지
├── doodle.php      # Doodle 아카이브 페이지
├── project.php     # Project 상세 페이지
├── resume.php      # Resume 페이지
└── contact.php     # Contact 페이지
```

## 배포

이 프로젝트는 PHP를 지원하는 웹 서버에 배포할 수 있습니다.

현재 라이브 배포 주소:

- [http://rohigraphy.co.kr:63478/](http://rohigraphy.co.kr:63478/)

기본 요구사항은 다음과 같습니다.

- PHP 지원 호스팅
- Apache 또는 Nginx 같은 웹 서버
- JSON 콘텐츠 수정을 위한 `data/` 쓰기 권한
- 이미지 업로드를 위한 `assets/images/uploads/` 쓰기 권한
- 폰트 업로드를 위한 `assets/fonts/uploads/` 쓰기 권한

## 향후 계획

현재는 단순성과 이식성을 위해 JSON 기반으로 동작합니다. 추후에는 웹 데이터베이스 기반 콘텐츠 시스템으로 전환할 예정입니다.

예정된 개선 방향은 데이터베이스 기반 콘텐츠 저장, 고급 미디어 라이브러리 관리, 향상된 필터링과 검색, 이미지 메타데이터 지원 강화, 콘텐츠 수정 버전 관리, 더 유연한 페이지 빌더 기능, 운영용 사이트맵과 SEO 워크플로우입니다.
